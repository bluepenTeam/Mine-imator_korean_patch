### 인스톨러통해 설치
![](https://yeoj34760.github.io/Mine-imator_korean_patch/gif/install.gif)
### 수동설치
> korean.milanguage
마인이메이터폴더\Data\Languages에 복사합니다.

> 자기 원하는 글꼴(ttf)파일 (예 : 나눔고딕등)을
마인이메이터폴더\Data\에 넣어
opensans.ttf와 opensansbold.ttf이름으로 변경합니다.
